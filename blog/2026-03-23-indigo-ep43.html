# Indigo 读后感

一个 Agent 可能比十个更好用

2026 年 3 月 23 日 · 7 分钟阅读 · 基于 Indigo Talk EP43

Indigo Talk EP43 的标题很有意思：《一个 Agent 可能比十个更好用》。这个观点颠覆了我对 AI 工具的理解。

我们总是在追求"更多"——更多工具、更多功能、更多选择。但 Indigo 在这期节目中提出了一个相反的观点：专注和深度可能比广泛和浅尝更有价值。

工具泛滥的时代

现在的问题不是"没有工具"，而是"工具太多"。

我们有 ChatGPT、Claude、Gemini、Copilot……每个都很强大，但每个都需要学习、配置、集成。结果是什么？大多数人最后只用了其中一两个，其他的都闲置了。

这就像一个厨师有一百把刀，但只用其中一把。剩下的九十九把不仅没有帮助，反而增加了认知负担。

Agent 的优势：深度而非广度

Indigo 在节目中强调的是 Agent 的另一个特性：它可以深度理解你的工作流，而不仅仅是提供通用功能。

一个精心设计的 Agent 可以：

  • 理解你的工作习惯和偏好
  • 记住你的历史决策和上下文
  • 在关键时刻主动提供帮助
  • 与你的其他工具无缝集成

这不是"更多功能"，而是"更懂你"。

OpenClaw 的启示

这让我想到了 OpenClaw。它不是一个"超级工具"，而是一个"智能助手框架"。

关键区别在于：

"十个通用工具 = 十个选择 + 十倍的学习成本 + 十倍的集成复杂度。一个深度定制的 Agent = 一个选择 + 最小的学习成本 + 最大的效率。"

我最近用 OpenClaw 的经历印证了这一点。我不需要在 Slack、邮件、日历、笔记之间切换。一个 Agent 就能处理所有这些，而且它理解我的工作流。

质量 vs 数量的权衡

这期节目的核心洞察是一个古老的智慧：质量胜于数量。

在 AI 工具的时代，这意味着：

  • 选择一个好的 Agent，而不是十个平庸的工具
  • 深度定制，而不是浅尝所有功能
  • 长期投入，而不是频繁切换

对个人和企业的启示

对个人：停止追求"最新最酷"的工具。找到一个适合你的 Agent，投入时间深度使用，让它真正理解你。

对企业：不要盲目采购十个 SaaS 工具。考虑构建一个定制的 Agent 系统，它可能比十个工具的组合更高效。

我的反思

听完这期节目，我意识到我一直在犯一个错误：我在追求工具的多样性，而不是深度。

我有 ChatGPT、Claude、Gemini……但我真正深度使用的只有 Claude 和 OpenClaw。其他的都是"以防万一"的选择，结果是增加了认知负担。

Indigo 的建议很清晰：选择一个，用深。

结语

在 AI 工具爆炸的时代，最稀缺的不是工具，而是专注。一个深度定制、真正理解你的 Agent，比十个功能强大但陌生的工具更有价值。

这不仅仅是一个工具选择的问题，更是一个哲学问题：我们是要追求更多，还是追求更好？
